# Led Blink whith Crontab

Este proyecto, es un led blink con pasos y estados, y disparado por una implementacion de un crontab.

El crontab es un timer periodico de 0.5 segundos, que valida si la tarea se debe ejecutar.

El led blink es un timer periodico de 1 segundo, que valida si se inicio la tarea y la ejecuta, al finalizar limpia las banderas.

Este proyecto, es parte las pruebas de exploracion para el sistema de riego.
